客製化Bootstrap4的方法(VScode/SCSS)

發表於 2020-02-16 | 分類於 bootstrap

Bootstrap用麥當勞來比喻像是一家店
想來一份麥香雞套餐就預設有漢堡薯條加飲料
也都可以做一些“微調”像是要漢堡不要生菜薯條去鹽飲料去冰
如果要做非常細緻的微調(例如糖度87%)就沒辦法了XD

一般使用Bootstrap不外乎就是用它現有的元件跟框架就可以輕鬆完成網頁
可是預設只有那幾種顏色～如果想改該怎麼辦呢
這時候就可以自定義的Bootstrap
讓你想怎麼改就怎麼改～完全客製化變成你的形狀

事前準備

  • 使用VSCode開發環境
  • 創建project資料夾新增index.html

    project/index.html

  • 具備scss的語言基礎

VSCode安裝2個套件

去VSCode左側工具欄尋找Extensions鍵入關鍵字搜尋

  • Live Sass Compiler (編譯scss用)
  • Live server (存檔後即時更新預覽用)
    安裝完畢後，VS Code底下出現兩個按鈕：watch sass 和 Go Live
    點擊並啟用

載入Bootstrap(3種方法選一種)

推薦使用npm安裝

  • CDN(雲端連結)
  • npm(使用終端機在本地端安裝) 安裝成功後會在project資料夾中出現node_modules資料夾

project/node_modules

  • Dowload(下載放本地端)
    官網下載位址

測試scss的運作

project資料夾內創建stylesheets資料夾+all.scss

project/stylesheets/all.scss

開啟all.scss隨便寫什麼測試一下後儲存
all.scss會自動編譯產生all.css和all.css.map
這時候就在index.html加入all.css

所有scss寫的東西都會轉換成css即時更新啦！！

進階使用bootstrap起手式

從這裡開始進入bootstrap

  • 主題化匯入Theming Bootstrap
    在all.scss內可以先丟全部載入測試運作是否成功 要客製化的話當然就要選部分載入囉！

接著在bootstrap裡隨便找一個元件丟進index.html去測試看有沒有執行成功

主題化我的bootstrap

主題化裏最重要的是 _variables.scss

所有的變數都在這裡改～牽一髮動全身
到這個路徑找到_variables.scss

在stylesheets內創建新資料夾helpers
把_variables.scss另存新檔到這裡

在all.scss把路徑改成自訂的helpers底下路徑後存檔

接著就可以改variables啦～～
